## Product Backlog

1. Como jogador, quero criar um nickname para me identificar no jogo.
2. Como jogador, quero iniciar um novo jogo de batalha naval.
3. Como jogador, quero jogar contra um robot para poder jogar sozinho.
4. Como jogador, quero criar um link de convite para jogar contra outro jogador.
5. Como jogador, quero consultar as instruções do jogo antes de começar.
6. Como jogador, quero visualizar o meu tabuleiro de jogo.
7. Como jogador, quero visualizar o tabuleiro do adversário.
8. Como jogador, quero atacar uma posição no tabuleiro adversário.
9. Como jogador, quero receber feedback quando o ataque acerta num navio.
10. Como jogador, quero receber feedback quando o ataque falha.
11. Como jogador, quero saber quando o jogo termina.
12. Como jogador, quero ver se ganhei ou perdi a partida.
